Manual:

In ORDER to execute the program the program should contain a file.txt with the name secret (“secret.txt”) and execute the python script (“VAULTmenu.py”) from the terminal where all the python files are located of the program. Therefore go to the the path with the file containing both the python files and secret.txt.
Note: when you encrypt a file you encrypt the first line. You can only select one file at a time.

You will be prompted to either use RSA or AES.
Follow the instructions from the program.
